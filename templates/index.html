📋 使用流程

  1. 选择上传方式（本地文件或URL）
  2. 上传文档到 Aitu Passport
  3. 点击生成的签名链接
  4. 在 Aitu Passport 中完成签名
  5. 签名完成后自动回调获取结果

🔌 API 端点

  • POST /upload - 上传本地文档
  • POST /upload-by-url - 通过URL上传文档
  • GET /sign?signable_id=xxx - 发起签名流程
  • GET /callback - OAuth2 回调处理

💻 命令行示例

上传本地文件：

curl -X POST -F "file=@document.pdf" http://localhost:8000/upload

通过URL上传：

curl -X POST -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/doc.pdf"}' \
  http://localhost:8000/upload-by-url
